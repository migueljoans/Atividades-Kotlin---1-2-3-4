# Portal do Candidato - ENEM

---

## Funcionalidades

* **Cadastro e Login:** Sistema de autenticação simulado utilizando o `localStorage` do navegador.
* **Painel Principal:** Dashboard com atalhos para as seções do portal.
* **Filtro de Cursos:** Listagem dinâmica de cursos baseada na área selecionada (Exatas, Humanas, Natureza e Linguagens).
* **Formulário de Inscrição:** Validação de campos e máscaras dinâmicas para CPF e Telefone via JavaScript.

---

## Tecnologias Utilizadas

* **HTML5:** Estruturação das páginas e formulários.
* **CSS3:** Design responsivo, estilização visual e uso da fonte *Outfit*.
* **JavaScript:** Lógica de validação, filtros dinâmicos e persistência de dados localmente.
